firewall指令

打开端口：firewall-cmd –permanent –add-port=端口号/协议
关闭端口：firewall-cmd –permanent –remove-port=端口号/协议
重新载入，才能生效：firewall-cmd –reload
查询端口是否开放：firewall-cmd –query-port=端口/协议
查询已经开放的端口号：firewall-cmd –zone=public –list-ports

Docker安装：

添加依赖库yum install -y yum-utils device-mapper-persistent-data lvm2
添加Docker源
yum-config-manager –add-repo https://download.docker.com/linux/centos/docker-ce.repo
yum-config-manager –add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装docker以及docker-cli
yum install docker-ce-18.09.0 docker-ce-cli-18.09.0 containerd.io
启动Docker
systemctl start docker
Docker开机自启
systemctl enable docker

在Docker中安装nginx

  1. 拉取nginx docker pull nginx:1.23.1

  2. 创建数据卷地址
    mkdir /data/nginx/conf
    mkdir /data/nginx/log
    mkdir /data/nginx/html

  3. 启动nginx ，然后copy初始配置文件到新建的文件目录下
    生成容器
    docker run –name nginx -p 9001:80 -d nginx:1.23.1
    将容器nginx.conf文件复制到宿主机
    docker cp nginx:/etc/nginx/nginx.conf /data/nginx/conf/nginx.conf
    将容器conf.d文件夹下内容复制到宿主机
    docker cp nginx:/etc/nginx/conf.d /data/nginx/conf/conf.d
    将容器中的html文件夹复制到宿主机
    docker cp nginx:/usr/share/nginx/html /data/nginx/html

  4. 删除容器
    docker rm nginx

  5. 重启容器并挂载数据卷
    docker run
    -p 9002:80
    –name nginx
    -v /data/nginx/conf/nginx.conf:/etc/nginx/nginx.conf
    -v /data/nginx/conf/conf.d:/etc/nginx/conf.d
    -v /data/nginx/log:/var/log/nginx
    -v /data/nginx/html:/usr/share/nginx/html
    -d nginx:1.23.1

  6. 打开防火墙9002端口并测试
    主机IP:9002
